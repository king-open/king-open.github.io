Hugo搭建教程

June 4, 2020

大家好

带来一篇windows用户 hugo搭建教程

hugo安装

windows用户64.zip或86.zip,并解压hugo.exe，放在自己的盘中 把D:xx/xxx/hugo加到PATH中，重启终端 运行hugo version 查看版本。

进入hugo官网点击快速开始，从Step2开始~Step7一步步copy，得到一个public，这就是博客的站点 请使用hugo server预览

Step2:

hugo new site quickstart (最后一个改成 自己的博客网址:xxx.github.io)

Step3:

cd xxx.github.io

git init

git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke


echo 'theme = "ananke"' >> config.toml

Step4:

创建第一篇博客

hugo new posts/开博大吉.md

draft: true 后面的true的意思是 草稿，改为false
其他不要改

Step5:

 hugo server -D

Step6:

code config.toml /**用code打开**/

baseURL = "https://example.org/"  /*不要改*/
languageCode = "en-us"  /*改成 zh-Hans*/
title = "My New Hugo Site" /*改成 xxxx 博客*/
theme = "ananke" /*主题 不要改*/

Step7:

hugo -D

部署：

先在 空白内创建一个.gitignore 在里面写入/public/,再进入public目录 在创建一个仓库命令：

git init

,再把public所有文件提交，后面加一个 空格一下 . ，.代表当前目录

 git add .

, 提交一下

 git commit -v

第一次部署。去github新建一个仓库 ，仓库名就叫你的github名.github.io,下面啥也不要选 用下面两个命令，然后上传。就搞定了。

刷新，再打开Settings往下拉有个会自动生成这个网址，打开

完成了
